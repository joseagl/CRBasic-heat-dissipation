This folder includes CRBasic scripts used to monitor differential voltage, which is then used to estimate sap flux density and sap flow using the heat dissipation method (Granier, 1984)\

The scripts were at some point part of a research project, and might include other sensors\

Sharing for reference purposes. User assumes all responsibility\

================================================================================================================

Files included:

CR1000_AM1632B.CR1: CRBasic script written to collect data using a CR1000 (outdated) data logger and a AM1632B multiplexor. Includes some soil moisture sensors, and a HMP50/HMP60 temperature & Relative Humidity sensor}
